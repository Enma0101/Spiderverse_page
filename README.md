# Comics Spider

Comics Spider is a web-based platform dedicated to the Spider-Man universe, designed to provide a rich, interactive experience for fans. This project focuses on professional-grade web development, utilizing modern technologies to deliver a seamless and engaging interface.

## Core Technologies

- **Frontend Framework**: React 19 provides the foundation for a responsive and performant user interface.
- **Build Tool**: Vite ensures a fast development environment and optimized production builds.
- **Styling**: Tailwind CSS 4 is used for a clean, modern design with a focus on utility-first CSS.
- **Backend Integration**: Supabase handles data storage and management, providing a robust backend-as-a-service solution.
- **Animations and 3D**: Framer Motion and AOS are used for smooth transitions, while Three.js integrates interactive 3D elements.
- **Icons**: Lucide React provides a comprehensive and consistent iconography system.

## Key Features

- **Interactive Gallery**: A visually appealing display of Spider-Man media and artwork.
- **Comic Reader**: A specialized interface for a high-quality reading experience.
- **Audio Integration**: A floating audio player for immersive background music.
- **Responsive Design**: Fully optimized for various devices, from desktop screens to mobile phones.
- **3D Elements**: Integration of 3D models to enhance the visual depth of the site.

## Development Setup

### Prerequisites

- Node.js (version 18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository to your local machine.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and configure your Supabase credentials.

### Running Locally

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
